## DICAS DE USO NO GPT QUADRINHOS SABENDO

- Gere sempre a folha de personagens primeiro e salve a imagem antes de prosseguir
- Ao gerar cada página, comece com "Character reference: same characters as previous page"
  (já incluído nos prompts das páginas 2, 3 e 4)
- Se um painel não sair com a Lis consistente, regenere apenas aquela página
- Os balões em português já estão incluídos nos prompts — não é necessário traduzir
- Cada bloco de código é um prompt completo — copie tudo dentro das marcações de backtick
